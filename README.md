## Kwitter : Twitter Clone Project

트위터 클론 프로젝트 

  
## 사용 기술

> Firebase : Firestore, Authentication, 

> React.js, SCSS

## 구현된 기능

* 게시글(트윗)
  * 댓글
  * 리트윗 

* 회원가입 및 로그인
  * Firebase Authentication을 활용한 회원가입 및 로그인 구현
  * Firebase Firestore 추가 회원정보 구현
    * 프로필
